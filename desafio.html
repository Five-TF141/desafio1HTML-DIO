1. Tags e seus atributos

  1. Comando de atalho para formatação base de HTML: html:5 + tab, dentro do VSC.
  2. Um id serve para criar um identificador para um elemento html.
  3. Input type: local para usuário inserir informações, tais como text para texto, number para um seletor de números, e color para abrir uma paleta seletora de cores.
  4. Atributos são colocados dentro das tags para personalizá-las e solicitar que se comportem da maneira solicitada.

2. Textos

  1. Headers (hnumero) funcionam como títulos e subtítulos, quanto menor o número, maior será a fonte.
  2. p serve para parágrafo.
  3. blockquote é uma tag usada para citações nas páginas; por padrão ela já vem com uma formatação propícia para isso.
  4. Tags úteis para texto: i itálico; strong negrito; u sublinhado; mark dá highlight no texto; sup coloca o texto um pouco acima da linha, como se fosse um expoente.
  5. br pula linha.
  6. hr traça uma linha horizontal (pode ser customizada com style).

3. Listas

  1. Lista ordenada se usa ol (ordered list), e para listas não ordenadas, ul (unordered list).
  2. Cada item da lista começa com a tag li.

4. Links

  1. Tag a serve para links junto com o atributo href, que contém o endereço de link.
  2. A estrutura seria a, href com o link e após, um texto que ao clicar abriria o link.
  3. Pode usar o target=˜_blank” para abrir em outra aba.
  4. Atributo title=”” serve para aparecer um texto enquanto o cursor estiver hovering acima do texto do link .
  5. Referenciar o nome de outro arquivo no href irá abri-lo em outra página web.
  6. Ancoragem: criar um id=”nomeID” dentro da tag destino, e a referenciar dentro do href de onde o usuário irá clicar. No href, referenciar a id a colocando entre aspas e com # na frente.
